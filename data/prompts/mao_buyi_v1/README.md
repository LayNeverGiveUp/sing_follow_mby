# Mao Buyi Prompt Audio Assets

Place licensed original vocal prompt clips in this directory.

Expected filenames:

- `mao_buyi_xiaochou_prompt.wav` for 消愁
- `mao_buyi_like_me_prompt.wav` for 像我这样的人
- `mao_buyi_summer_prompt.wav` for 盛夏
- `mao_buyi_unstained_prompt.wav` for 不染
- `mao_buyi_ordinary_day_prompt.wav` for 平凡的一天
- `mao_buyi_meat_and_vegetable_prompt.wav` for 一荤一素
- `mao_buyi_borrow_prompt.wav` for 借
- `mao_buyi_murmur_prompt.wav` for 呓语
- `mao_buyi_no_question_prompt.wav` for 无问
- `mao_buyi_muma_city_prompt.wav` for 牧马城市

Use short clips for the browser demo, ideally 3-8 seconds each. The UI will
play these files from `/static/prompts/mao_buyi_v1/<filename>`.

If a file is missing, the UI falls back to a synthetic melody prompt for that
song so testing can continue.
